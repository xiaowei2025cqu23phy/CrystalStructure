## 🧪 科学背景：什么是晶体结构？

在化学和物理学中，**晶体结构**是指原子、离子或分子在空间中高度有序的排列方式。

*   **晶格 (Lattice)**：晶体中原子排列的抽象几何框架。
*   **晶胞 (Unit Cell)**：能够代表晶体结构特征的最小重复单元。通过在三维空间中平移晶胞，可以构建出整个晶体。
*   **点阵类型**：
    *   **SC (简单立方)**：原子位于立方的八个顶点。
    *   **BCC (体心立方)**：顶点有原子，立方体中心还有一个原子（如 铁、钨）。
    *   **FCC (面心立方)**：顶点有原子，每个面的中心也有一个原子（如 铜、铝、金）。
    *   **HCP (六方最密堆积)**：原子以六棱柱形式排列，空间利用率极高（如 镁、钛）。
*   **分子结构**：与无限延伸的晶体不同，分子（如 C60 富勒烯或 H2O 水分子）具有特定的几何形状和有限的原子数量。

**本工具的作用**：通过 3D 建模，将这些抽象的数学描述转化为直观的视觉模型，帮助用户理解原子是如何在微观世界中“搭建”起宏观物质的。

## 🌟 核心功能

### 1. 🔬 深度 3D 可视化
- **双视图并行**：左侧显示带晶胞边界的“结构单元”，右侧显示无边界的“纯净点阵”，方便对比微观对称性。
- **智能渲染**：自动识别晶体（无限延伸）与分子（独立个体），并根据结构特性优化相机视角。
- **动态成键**：实时计算原子间距，并根据自定义阈值（Å）渲染化学键。

### 2. 📊 实时物理计算
- **几何参数**：即时计算晶胞体积（Å³）。
- **材料特性**：根据原子量和晶格常数计算理论密度（g/cm³）及空间利用率（%）。
- **对称性矩阵**：展示晶体学中的对称操作矩阵及其物理含义。

### 3. ⚛️ 交互式元素周期表
- **全属性展示**：内置 H 到 Rn 的详细数据，包括原子序数、原子量、电负性。
- **电子层排布**：**独家功能**，展示元素的标准电子排布（如 Fe: `[Ar] 3d⁶ 4s²`）。

### 4. 📤 专业级导出
- 支持一键导出 **PNG (高质量位图)** 或 **SVG (科研级矢量图)**。

## 💻 电脑端运行指南

如果您想在自己的电脑上运行此项目，请遵循以下步骤：

### 1. 环境准备
确保您的电脑已安装：
*   **Node.js** (推荐 v16.14.0 或更高版本)
*   **npm** (通常随 Node.js 一起安装)

### 2. 获取代码
如果您是从 AI Studio 导出的项目：
1.  解压下载的 `.zip` 文件到您的工作目录。
2.  打开终端（Windows 使用 PowerShell 或 CMD，Mac/Linux 使用 Terminal）。

### 3. 安装依赖
在项目根目录下运行：
```bash
npm install
```
这会自动下载并安装 React、Three.js、Tailwind CSS 等所有必要的库。

### 4. 启动开发服务器
运行以下命令：
```bash
npm run dev
```
终端会显示一个链接（通常是 `http://localhost:3000`）。在浏览器中打开该链接即可看到应用。

### 5. 构建生产版本
如果您想部署到服务器：
```bash
npm run build
```
生成的静态文件将存放在 `dist` 文件夹中。

## 📱 手机端查看与部署

本项目针对移动端进行了深度优化，您可以随时随地展示您的晶体模型。

### 1. 直接查看（最快方式）
如果您已经有了部署好的链接（如 AI Studio 提供的预览链接）：
*   直接在手机浏览器（Safari, Chrome, 夸克等）中打开链接。
*   **操作手势**：
    *   **单指滑动**：旋转模型。
    *   **双指捏合**：缩放模型。
    *   **三指滑动**：平移视角。

### 2. 局域网同步查看
如果您在电脑上运行着开发服务器，且手机与电脑在**同一个 Wi-Fi** 下：
1.  在电脑终端查看您的局域网 IP（Windows 输入 `ipconfig`，Mac 输入 `ifconfig`）。
2.  在手机浏览器输入 `http://<您的电脑IP>:3000`。

### 3. 手机端本地部署 (Android)
Android 用户可以使用 **Termux** 尝试真正的本地运行：
1.  安装 Termux 应用。
2.  执行 `pkg install nodejs git`。
3.  克隆代码并按照“电脑端运行指南”执行 `npm install` 和 `npm run dev`。
